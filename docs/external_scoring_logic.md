# External Scoring Methodology: STAR Impact Ranking

This document outlines the public-facing scoring methodology for the STAR Impact Ranking. This rubric is visible to all users on the rankings page and defines how institutions are evaluated based on their participation in the STAR ecosystem.

## External Scoring Categories (Visible to Public)

The final **STAR Impact Score** (out of 100) is derived from the following four external categories.

| Category | Weight | Description (Visible in Rubric) |
| :--- | :--- | :--- |
| **Total Circle Participants** | 30% | How many circles have individuals from that institution/university participated in. |
| **Total Circles Run** | 10% | How many circles have been run in the past, are actively running, and will run in the future led by individuals from that university. |
| **STAR Awards Won** | 30% | How many STAR affiliated awards (especially the Global Education for Peace award) have been won by that university. |
| **STAR Circle Engagement** | 30% | What was the circle engagement rate among the individuals from a certain university. |

## Table Structure

The official rankings table displays the following columns:

1.  **Rank**: The institution's position in the global index.
2.  **Institution Name**: The full name of the university/organization.
3.  **Country**: The primary country where the institution is based.
4.  **Total Circle Participants**: Count of participants.
5.  **Total Circles Run**: Count of circles led/run.
6.  **STAR Circle Engagement**: The engagement rate.
7.  **STAR Impact Score**: The final calculated score (0-100).

> [!NOTE]
> The STAR Impact Score is a comprehensive metric designed to highlight institutions excelling in transnational research, community engagement, and scholar support.
